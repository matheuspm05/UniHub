Módulo Fórum — visão geral

A estrutura principal é:

Fórum
├── Cursos
├── Disciplinas
├── Tópicos
└── Respostas

E o fluxo base:

Usuário entra no Fórum
        ↓
Vê tópicos do próprio curso primeiro
        ↓
Filtra por curso/disciplina/categoria
        ↓
Abre um tópico
        ↓
Responde ou cria novo tópico

2. Telas do moderador no Fórum

Aqui tem um ponto importante.

No projeto de vocês, o moderador é mais um representante/organizador autorizado, não necessariamente um administrador total do fórum.

Então eu faria o moderador assim:

Moderador pode:
Criar tópicos normalmente
Responder tópicos
Editar os próprios tópicos
Criar avisos com selo especial
Ter selo de Moderador/Representante
Moderador não pode:
Editar tópico de qualquer usuário
Apagar resposta de qualquer usuário
Bloquear usuário
Gerenciar todo o fórum

Isso fica para o admin.

Telas específicas do moderador

Na prática, ele usa quase as mesmas telas do usuário comum, mas com pequenas diferenças.

1. Página principal do Fórum — versão moderador

Diferenças:

Perfil com selo: Moderador
Botão: + Novo tópico
Botão opcional: + Criar aviso

Abas possíveis:

[Explorar fórum] [Meus tópicos] [Avisos]
2. Criar tópico / Criar aviso

O moderador pode criar um tópico normal ou um aviso.

Campo especial:

Tipo de publicação:
- Tópico comum
- Aviso oficial

Exemplo:

Aviso: Monitoria de Banco de Dados nesta sexta

Esse aviso apareceria com selo:

Aviso oficial
3. Meus tópicos do moderador

Mostra os tópicos e avisos criados por ele.

Ações:

[Editar]
[Fechar tópico]
[Marcar como resolvido]
Telas do moderador no Fórum
1. Página principal do Fórum — versão moderador
2. Criar tópico / criar aviso
3. Detalhe do tópico com selo de moderador
4. Editar tópico próprio
5. Meus tópicos / meus avisos
3. Telas do admin no Fórum

O admin é quem tem controle total do fórum.

No documento, o administrador pode gerenciar tópicos e respostas do fórum, além de desativar conteúdos inadequados e bloquear usuários se necessário.

Admin pode:
Ver todos os tópicos
Editar qualquer tópico
Fechar qualquer tópico
Remover/desativar tópicos inadequados
Remover/desativar respostas inadequadas
Gerenciar categorias, cursos e disciplinas
Bloquear usuário, se necessário

Estrutura final do módulo Fórum

Eu organizaria assim:

Fórum
├── Explorar fórum
├── Detalhe do tópico
├── Criar tópico
├── Editar tópico
├── Meus tópicos
├── Minhas respostas
├── Criar aviso — moderador/admin
├── Gerenciar tópicos — admin
├── Gerenciar respostas — admin
└── Modal de confirmação
Para o MVP, eu faria primeiro

Para não ficar gigante demais, a ordem ideal é:

1. Página principal do Fórum
2. Detalhe do tópico
3. Criar tópico
4. Editar tópico
5. Meus tópicos / minhas respostas

Depois:

6. Versão moderador com aviso oficial
7. Gerenciar tópicos — admin
8. Gerenciar respostas — admin
Resumo direto

O fórum é o coração social/acadêmico do UniHub.

Ele precisa parecer uma mistura de:

Discord acadêmico
+
Stack Overflow simplificado
+
mural de curso

Mas sem complicar demais.
